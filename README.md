# toska30.github.io

Personal landing page — published at [toska30.github.io](https://toska30.github.io).

Single HTML file, embedded CSS, no build step. Serif headings, sans body, plenty of whitespace. Optimized for readability and being honest about what I do.

## Local preview

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000.

## Deployment

GitHub Pages serves this automatically from the `main` branch.
